# Frontend - Product Management

A React application for managing products, built with Vite.

## Features

- Create, Read, Update, and Delete products
- Modern and clean UI
- Built with React and Vite
- Uses `axios` for API requests

## Prerequisites

- Node.js and npm installed
- The backend server from the `03_python_fastapi_project` must be running.

## Installation

1.  Navigate to the `04_ai_coding_assistants/frontend` directory.
2.  Install the dependencies:
    ```bash
    npm install
    ```

## Running the Application

1.  Make sure the backend server is running.
2.  Start the development server:
    ```bash
    npm run dev
    ```
3.  The application will be available at `http://localhost:5173`.

## Building for Production

To create a production build, run the following command:

```bash
npm run build
```

This will create a `dist` directory with the optimized and minified files.
